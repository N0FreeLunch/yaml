## YAML 이란
- 데이터 직렬화 언어

### 직렬화란?
- 나중에 재구성할 수 있는 포맷으로 변환하는 과정
- 마샬링이라고도 부른다.

### 특이 문법 요소
- 들여쓰기 : 데이터의 계층화 구조화
- 콜론 : 키/벨류의 구분
- 대시 : 글머리기호(•)가 붙은 리스트

### 쓰임
- 환경 설정 파일
- 로그 파일
- 프로세스간 메시징
- 언어 간 데이터 공유
- 객체를 영속적 저장소에 저장
- 복잡한 데이터 구조의 디버깅

## 특징
- (우연의 일치이지만) JSON은 YAML의 하위집합임
- 이후 엄격하게 YAML을 JSON의 상위 집합으로 만들고자 함
